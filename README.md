# COVID-19 Interactive Dashboard

An interactive public health dashboard exploring **COVID-19 confirmed cases, deaths, recovery trends, and WHO regional differences** using Python, Plotly, and Dash.

---

## Project Overview

This project combines **data wrangling, exploratory data analysis (EDA), interactive visualizations, and dashboard development** to analyze COVID-19 trends across countries and WHO regions.

The dashboard enables users to:

✅ Filter by **WHO region**  
✅ Adjust **death count thresholds**  
✅ Explore relationships between **confirmed cases and deaths**  
✅ Compare trends across countries and regions

The project also includes additional visualizations and statistical summaries to better understand global COVID-19 outcomes.

---

## Methods Used

### Data Preparation
- Data cleaning and transformation
- Column standardization
- Missing data handling
- Feature engineering

### New Metrics Created
- **Case Fatality Rate (CFR)**
- **Recovery Rate**

### Data Analysis
- Regional aggregation
- Comparative country analysis
- WHO regional trend analysis

### Visualizations
- Interactive Scatterplots
- Interactive Line Charts
- Recovery Rate Rankings
- Boxplots
- Faceted Visualizations
- Density Distributions
- Interactive Dashboard (Dash + Plotly)

The dashboard was built using **Python Dash callbacks**, allowing dynamic filtering based on WHO region and death thresholds. 

---

## Key Findings

📌 Countries with higher confirmed cases generally reported higher death counts.

📌 **The Americas and Europe showed the highest case and death totals**, while Africa and the Western Pacific had lower overall burdens. 

📌 Most countries clustered between **0–5% case fatality rate**, with many concentrated around **2–3%**.

📌 **Italy and the United States demonstrated the highest death burden relative to confirmed cases among the top 5 countries analyzed.**

📌 Countries with more **new cases tended to report more new deaths**, particularly in heavily affected regions.
